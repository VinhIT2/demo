# FINAL2026_LayoutNgang

Ứng dụng quản lý đơn vận chuyển theo mẫu layout ngang.

## Cấu trúc

- `html/`: bản HTML/CSS/JS thuần.
- `reactjs/`: bản ReactJS tách component.

## Chức năng

- Hiển thị dữ liệu động từ `data.js`.
- Tìm kiếm và lọc theo trạng thái theo giao diện mẫu.
- Thêm mới đơn vận chuyển.

Không có sửa/xóa vì mẫu đề không yêu cầu.
