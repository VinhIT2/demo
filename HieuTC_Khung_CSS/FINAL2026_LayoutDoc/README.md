# FINAL2026_LayoutDoc

Ứng dụng quản lý hoạt động ngoại khóa theo mẫu layout dọc.

## Cấu trúc

- `html/`: bản HTML/CSS/JS thuần.
- `reactjs/`: bản ReactJS tách component.

## Chức năng

- Hiển thị dữ liệu động từ `data.json`.
- Tìm kiếm và lọc theo trạng thái theo giao diện mẫu.
- Thêm mới hoạt động.

Không có sửa/xóa vì mẫu đề không yêu cầu.

Lưu ý: bản HTML dùng `fetch("./data/data.json")`, nên nên chạy bằng Live Server/local server thay vì mở file trực tiếp.
